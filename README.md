# MERN Authentication System

## Features

- User Registration
- Login
- JWT Authentication
- Email Verification OTP
- Forgot Password
- Reset Password
- Protected Route
- Logout

## Tech Stack

- React
- Express
- Node.js
- MongoDB
- JWT
- Bcrypt
- Nodemailer

## Installation

Backend

npm install

Frontend

npm install

Run Backend

npm start

Run Frontend

npm run dev
